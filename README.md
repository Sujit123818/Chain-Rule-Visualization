# Chain-Rule-Visualization
 The chain rule isn't something you memorize it falls out naturally from how changes combine.
 Step           Meaning                                                                                   
 ------------- ----------------------------------------------------------------------------------------- 
 **Steps 1-3**  Changes **propagate** through intermediate variables—each change multiplies with the next 
 **Step 4**     We rearrange to see the ratio of total change to initial change                           
 **Step 5**     As changes shrink to zero, these ratios become **derivatives**—and the rule still holds!
